# Personal Portfolio Website

## Executive Summary
This project is a personal portfolio website developed and deployed using **GitHub Pages**.  
It serves as a professional online presence to showcase academic background, technical skills, and completed projects.

The portfolio demonstrates basic web development capabilities and the ability to deploy and manage a live website using modern version control tools.

## Purpose of the Portfolio
The purpose of this portfolio is to:
- Present my profile as an upcoming IT professional
- Highlight academic background and interests
- Showcase selected academic and personal projects
- Provide clear contact and professional links
  
## Technologies Used
- **HTML** – Website structure and content
- **CSS** – Styling and layout
- **JavaScript** – Basic interactivity
- **GitHub Pages** – Free web hosting and deployment

## Key Features
- **Home Section**: Displays name, profile picture, and brief introduction
- **About Section**: Academic background and professional interests
- **Projects Section**: List of selected coursework and personal projects
- **Contact Section**: Email address and GitHub profile link

## Deployment
The website is hosted live using **GitHub Pages**, a free and reliable hosting service provided by GitHub.


## Value Demonstrated
This project demonstrates:
- Basic web development skills
- Use of version control with GitHub
- Ability to deploy and manage a live website
- Professional presentation of personal and academic information
